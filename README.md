# 🐉 Dungeon Adventure  
> 🎮 클래식한 감성의 턴제 콘솔 RPG

---

## 📖 소개

**Dungeon Adventure**는 플레이어가 던전에 들어가 **몬스터와 차례대로 전투**하며 점점 강해지고,  
마침내 **보스 몬스터를 처치하면 클리어**되는 간단한 텍스트 기반 **턴제 RPG**입니다.

---

## ⚔️ 게임 특징

- 🌀 **턴 기반 전투 시스템**  
  - 플레이어 턴 → 몬스터 턴 순으로 진행됩니다.
  - 매 선택이 생존과 직결됩니다.

- 💪 **성장 시스템**  
  - 몬스터를 처치하고 경험치를 얻어 레벨업!  
  - 레벨업 시: `HP +10`, `공격력 +2`

- 👑 **보스 몬스터**  
  - 특정 던전 단계마다 등장하는 강력한 적  
  - 보스를 처치하면 게임 클리어!
